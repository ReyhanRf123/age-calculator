# Age Calculator (HTML, CSS, JavaScript + Luxon)

This project is a **simple age calculator** built using **HTML, CSS, and JavaScript**, with the **Luxon** library for accurate date calculations.  
Users can select their birthdate and calculate their exact age in **years, months, and days**.

##  Features
- Custom JavaScript date picker (day, month, year)
- Accurate age calculation using Luxon
- Displays age in years, months, and days
- Basic input validation
- Responsive and simple UI
- No HTML default date input

## Learning Objectives
- Using external libraries (Luxon)
- Working with dates and time calculations
- Handling form submission with `preventDefault()`
- Validating user input
- Updating the DOM dynamically

## How It Works
1. User selects day, month, and year
2. Form submission is handled with JavaScript
3. Input is validated for correctness
4. Luxon calculates the exact age
5. Result is displayed on the same page

## How to Run
1. Open `index.html` in a browser
2. Select your birthdate
3. Click **Calculate Age** to see the result
